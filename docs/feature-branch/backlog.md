☐ Фильтрация строк с нулевыми значениями для выбранной группы/ресурса. Проектов много, но значимых строк бывает менее десятка — не в кайф скролить пустые строки
☐ На странице "Ресурсный план", столбец "доступно" не информативен. Лучше указывать часы, которые на данный момент доступны
☐ На вкладки "Ресурсный план" в столбце "Проект" добавить возможность выбора определенных проектов (выбор осуществляет пользователь) для отображения только их
☐ На вкладке "Ресурсный план", при наведении на проект выскакивает хинт, в нем помимо заказчика еще указывать и руководителя проекта.
☐ Автоматическое сохранение введенных данных на странице "Управление данными", без нажатия кнопки "Сохранить"
☐ Предлагается ввести помесячную фиксацию актуальности ресурсного плана внутри каждого квартала: руководитель проекта вручную подтверждает, что данные за конкретный месяц актуальны, с сохранением даты и пользователя подтверждения. При любых изменениях в ресурсах за соответствующий период статус автоматически сбрасывается на «не подтверждено», чтобы исключить использование устаревших данных. Это позволит руководителям подразделений видеть реальную актуальность планов в разрезе месяца.
☐ Рассмотреть целесообразность добавления валидации указанных РП ресурсов руководителями групп ресурсов. Например виза "Подтверждаю", которую может устанавливать/снимать руководитель, которая будет сбрасываться при актуализации плана РП. У значения запрошенного ресурса и у визы должны фиксироваться даты последнего изменения.
☐ Сейчас ресурсный план по некоторым целям не даёт понять на что именно планируется потратить ресурсы. Решение — рассмотреть возможность создания в confluence на странице каждой цели года укрупнённого плана проекта со списком задач-вех (разделённых по группам ресурсов) и оценивать планируемые ресурсы на эти задачи. Тогда в ресурсный план можно будет просто подгружать автоматически сумму по указанным целям года, а не проставлять вручную.
☐ Фильтрация проектов на странице "Управление данными" по активным проектам, по РП
☐ На вкладке Ресурсный план добавить маленькую кнопочку Актуализировано. РП должен ее нажать когда актуализирован ресурсы по проекту. И при наведении показывает дату последнего нажатия (соответственно нажимать эту кнопку должны быть права только у руководителей проектов) и каждое 20 число месяца актуализация слетает. Можно также сделать что бы у нее менялся цвет, что бы в было более явно видно кто актуализировал, а кто нет
